# age-structured-invasion-dynamics
Numerical solver and analysis scripts for Abo &amp; Baker (2025): Survival and invasion dynamics in cell populations.

## Experimental data for parameter fitting
We thank Rachel N. Leander for providing access to the DMSO, CHX, and ERL datasets available at https://github.com/rnleander/DDT\_cell\_cycle.
Users need to download the datasets from the repository in order to run NoDiffusionFigures.ipynb.
